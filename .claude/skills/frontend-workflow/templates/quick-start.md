# 🚀 前端工作流快速开始模板

复制这个模板，开始你的项目！

---

## 阶段 1：需求分析

- [ ] 收集所有需求点
- [ ] 与用户确认理解
- [ ] 确定优先级排序

---

## 阶段 2：制定计划

调用：
```
Use Skill: writing-plans
```

创建书面计划：
- [ ] 项目概览
- [ ] 文件变更清单
- [ ] 任务分解
- [ ] 验收标准

---

## 阶段 3：开发执行

调用：
```
Use Skill: subagent-driven-development
```

- [ ] 每个任务独立完成
- [ ] 修改前先读取文件
- [ ] 最小修改原则
- [ ] 修改后立即测试
- [ ] 每个修改单独提交
- [ ] 提交信息规范

---

## 阶段 4：全面测试

调用：
```
Use Skill: qa
Use Skill: webapp-testing
```

- [ ] 功能测试
- [ ] 视觉测试
- [ ] 响应式测试
- [ ] 控制台检查（零错误）

---

## 阶段 5：收尾提交

调用：
```
Use Skill: finishing-a-development-branch
```

- [ ] Git 状态检查
- [ ] 分支清理
- [ ] 最终确认
- [ ] 文档更新

---

## 完成！🎉

恭喜你完成了项目！现在可以开始下一个功能了！🚀
