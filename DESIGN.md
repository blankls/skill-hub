# Skill Hub 设计系统

## 🎨 设计理念
赛博朋克/黑客美学 - 技术感、未来感、功能性

## 📐 设计原则
1. **功能性优先** - 界面元素必须有明确用途
2. **视觉层次** - 通过颜色、大小、位置建立清晰的优先级
3. **一致性** - 相同的功能使用相同的视觉语言
4. **反馈明确** - 每个用户操作都有对应的视觉/交互反馈

## 🌈 颜色系统

### 主色调
```css
--neon-cyan: #00f5ff;    /* 赛博青 - 主要强调色 */
--neon-pink: #ff006e;    /* 霓虹粉 - 警告/危险 */
--neon-purple: #8338ec;  /* 霓虹紫 - 次要强调 */
--neon-yellow: #ffbe0b;  /* 霓虹黄 - 信息/提示 */
```

### 背景色
```css
/* 暗色模式 */
--dark-bg: #0a0e27;      /* 主背景 */
--dark-card: #141b3a;    /* 卡片背景 */

/* 亮色模式 */
--light-bg: #f8fafc;     /* 主背景 */
--light-card: #ffffff;   /* 卡片背景 */
```

### 文本色
```css
--text-light: #f0f4ff;   /* 主要文本 */
--text-muted: #94a3b8;   /* 次要文本 */
```

## 🔤 字体系统

### 字体族
```css
--font-mono: 'Courier New', monospace;  /* 主要字体 - 代码感 */
--font-sans: system-ui, sans-serif;     /* 辅助字体 - 可读性 */
```

### 字号层级
- **H1**: 3rem (48px) - 页面标题
- **H2**: 2rem (32px) - 区块标题
- **H3**: 1.5rem (24px) - 卡片标题
- **Body**: 1rem (16px) - 主要文本
- **Small**: 0.875rem (14px) - 辅助信息
- **XS**: 0.75rem (12px) - 标签/时间

## 📏 间距系统

使用 4px 基础单位：
- **XS**: 0.25rem (4px)
- **SM**: 0.5rem (8px)
- **MD**: 1rem (16px)
- **LG**: 1.5rem (24px)
- **XL**: 2rem (32px)
- **2XL**: 3rem (48px)

## 🎯 组件设计规范

### 按钮
- **主要按钮**: 霓虹渐变背景 + 霓虹边框 + 悬停发光
- **次要按钮**: 透明背景 + 霓虹边框
- **危险按钮**: 霓虹粉背景

### 卡片
- 毛玻璃背景 (`backdrop-blur`)
- 霓虹边框 (0.3 透明度)
- 悬停上移 + 发光效果
- 顶部彩色渐变条

### 输入框
- 深色背景
- 霓虹边框
- 聚焦时边框变亮

### 标签
- 圆角胶囊形状
- 对应颜色的半透明背景
- 前缀 # 符号

## 📱 响应式断点
- **移动**: < 768px
- **平板**: 768px - 1024px
- **桌面**: > 1024px

## ♿ 无障碍设计
- 触摸目标 ≥ 44px
- 文本对比度 ≥ 4.5:1
- 键盘导航支持
- ARIA 属性完整

## 🎬 动画规范
- 过渡时间: 0.3s ease (默认)
- 悬停动画: 轻微缩放 + 发光
- 加载动画: 旋转霓虹图标
